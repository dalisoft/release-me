## Plugins

| Name             | Description                                      |
| ---------------- | ------------------------------------------------ |
| `git`            | Creates **Git** tag and push to origin           |
| `github-release` | Release a tag with proper `CHANGELOG` and commit |
| `npm`            | Updates version field and Publishes **npm** tag  |
| `changelog`      | Generates `CHANGELOG` within your project        |
